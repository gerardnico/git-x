# git-auto-pull

## DESCRIPTION

Integrate the changes from an upstream
(Stash if needed and pull)

## SYNOPSIS
```bash
git auto-pull
```
AutoPull to stay in the same fashion than AutoCommit
## TIP

You can add it as alias in your `~.gitconfig`
Example with `sp` that stands for `stash, pull`
```ini
[alias]
sp = "!git-auto-pull"
```
