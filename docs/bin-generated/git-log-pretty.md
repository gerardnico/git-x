% git-log-pretty(1) Version Latest | git-log-pretty
# NAME

Enhance the default [medium pretty format](https://git-scm.com/docs/git-log#Documentation/git-log.txt-formatpretty)


## SYNOPSIS

```bash
git-log-pretty [args]
```
where `args` are arguments of `git logs`
