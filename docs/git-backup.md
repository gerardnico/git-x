# Git-Hosting - Backup your GitHub repositories

Don't get lockup of your Git Hosting repository if you lose your account.

Create backups with [git hosting backup](commands/git-hosting-backup.md)
so that you are always in control.

## Commands

* [git hosting](commands/git-hosting.md) - Main entry for Git Hosting Command
* [git hosting backup](commands/git-hosting-backup.md) - Backup your repositories on a Git Hosting Service
  such as GitHub
