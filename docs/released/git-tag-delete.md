% git-tag-delete(1) Version 0.1.0 | git-tag-delete

# NAME

Delete a tag (locally and remotely) and any GitHub release if any

## TIP

You can add it as alias

* in your `~/.gitconfig`

```ini
[alias]
    td = "git-tag-delete"
```

* in your `~/.bashrc`

```bash
alias gtd = "git-tag-delete"
```

## SYNOPSIS

```bash
git-tag-delete tags
```
tags is one or more tags separated by space
